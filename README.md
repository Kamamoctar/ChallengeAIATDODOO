# 📦 Pack Handover — Concours « App de saisie Odoo au quotidien »
## Togo Data AI Lab — pour les BINÔMES

Bienvenue ! Ce pack contient tout ce qu'il faut pour que **votre binôme** (2
personnes) construise, avec l'aide d'une IA, une application qui facilite le
**remplissage quotidien d'Odoo**.

Vous n'avez **aucun identifiant Odoo à gérer**, **aucune clé à créer**, **aucune
configuration manuelle** : l'IA fait tous les branchements toute seule.

## 🚀 Comment démarrer (2 minutes)

1. Ouvrez votre assistant IA (Claude, ChatGPT, Claude Code, Cursor…).
2. Ouvrez le fichier **`00_PROMPT_IA.md`** et **copiez-collez tout son contenu**
   dans l'IA.
3. L'IA va :
   - vous **demander vos prénoms + noms** (les 2 membres) ;
   - vous **retrouver automatiquement dans Odoo** ;
   - **brancher** l'accès aux données et vous aider à construire votre app.

C'est tout. Tout est déjà connecté. 🎉

## 📁 Contenu du pack

| Fichier | À quoi ça sert |

| **00_PROMPT_IA.md** | 👉 **LE fichier à coller dans votre IA.** Commencez ici. |

| **exemples/client.js** | Client JavaScript/Node prêt à l'emploi. |
| **exemples/requetes.http** | Requêtes de test (extension VS Code « REST Client »). |

## ℹ️ Bon à savoir

- La connexion à Odoo passe par une **passerelle sécurisée** ; le mot de passe
  Odoo n'est jamais exposé, ni à vous ni à l'IA. C'est voulu.
- La clé d'accès fournie est **partagée par tout le concours** : gardez-la hors
  de votre code (variable d'environnement), ne la publiez pas.
- Vous travaillez sur une base **partagée** : restez dans le périmètre de votre
  binôme et soyez prudents en écriture.

Bonne chance ! 🚀
